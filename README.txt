This plugin providing functionality for manage music albums

== Description ==

You can manage music albums in admin panel and you can display it on front-end using `[music]` shortcode. This plugin is providing functionality for manage music albums with music genres and music tag. Plugin has some inbuilt music meta information like Composer Name, Publisher, Year of recording, Additional Contributors, URL and Price.

Plugin is only providing listing page of music using `[music]` shortcode. User will be able to apply music genre and year filter in shortcode like `[music year="1993"]` or `[music genre="rock"]` or `[music year="1993" genre="rock"]`

== Installation ==

1. Upload my-music plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[music]` shortcode in your content
